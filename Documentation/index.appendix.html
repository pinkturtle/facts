Appendix

Internals

Facts relies on Immutable for its underlying data models.

Facts currently uses a modified copy of Backbone.Events which provides on and off methods to listen for completed transactions. This will probably be re-visited in the future.

Forecast

Version 0.0.1 will be the first release.

This turtle anticipates that the first release will occur after we complete the following hopeless tasks:

  • Specify query output format with Array, Object, Immutable.Map or Immutable.List instead of the existing string identifiers.
  • Remove the history property and replace with a function that constructs history-like instances on demand.
  • Document Facts.transact(facts, input).
  • Document Facts.database(facts, time).
  • Complete checklist example.
  • Complete the income tax example.
  • Complete the scene improv example.
  • Complete the tree building example.
  • Complete the puzzle example.
  • Complete the glossary example.
  • Optimistically, complete the traffic policy anaysis example.
  • Possibly include basic storage models for automatic sync with localStorage in a window or the file system on a node when "storage keys" are defined.

Radix

This turtle’s approach to information design was shaped early on by a CBC Radio broadcast of Ursula Franklin’s classic lecture about The Real World of Technology. Facts is a return to these roots because i think it is a demonstration of technology derived from humble practices that offers a holistic way of doing things rather than a prescription for how things should be done. Facts is meant to be enlightening, explorable, and without too much stress.

The programming model in Facts was copied from Datomic – especially its Clojure API. i first learned of Datomic while watching Rich Hickey present The Value of Values. His comparisson of place-oriented programming with value-oriented programming and his observations about space-age information systems struck a chord that continues to resonate. If you’re unfamiliar with Mr. Hickey, i recommend starting with this excerpt of him speaking about memory and records in society.

Facts was extracted from several information software prototypes that were inspired by Bret Victor’s wondrous yearnings for a dynamic media. The Humane Representation of Thought captures some of his most intriguing ideas on this topic and i highly recommend starting with this excerpt of him speaking about the inhumanity of our thinking media for a gentle introduction to his work. Mr. Victor is also an accomplished magician, puppeteer, life-coach and author.

This turtle is inclined to think the dynamic media should be playful. And since play happens over time, the media might rely on a time-based model of data to compute informative representations. Or maybe that’s already happening.
